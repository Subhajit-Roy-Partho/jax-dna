# jax-dna



### Development

`tox` is used to:
- Run tests
- Run the formatter
- Run the linter

To start with development create a python 3.11 environment and install `tox`, `python -m pip install tox`

1. To run tests: `tox -e test`
2. To run the formatter: `tox -e format`
3. To run the linter: `tox -e check-style`
